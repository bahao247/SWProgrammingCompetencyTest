[2017/1st]Fine Dust Measurement System Submit completed

2Second10points

*Solution (click): C++, C, Java

Standard Library reference : C++, C, Java


Home Appliance &Air solution Company's Air Conditioning Team intends to develop an air purification system that measures the fine dust content in the air to cope with the fine dust change which has been the key social issue currently. 


[Requirement]

The program that measures the change in fine dust particles hourly from hour 1 to hour N each day from day 1 to day N and extracts the day that shows the largest sum of changes during the total measured days and the time slot that shows the smallest sum of changes during each measurement day was generated. 


Using the following data of 4 hours for each of 4 days, the generated program runs as follows:





Output ‘3’ for the day that shows the largest sum of changes since the day 3 shows the largest change of 77.


The sum of changes on day 1 is 24 (2-15-15+52).

The sum of changes on day 2 is 47 (50+28+29-60).

The sum of changes on day 3 is 77 (-89+71+54+41).

The sum of changes on day 4 is -85 (-12-55-76+58).


Output ‘1’ for the time slot that shows the smallest sum of changes during each measurement day since hour 1 has the lowest value of -49.


The sum of changes at hour 1 is -49 (2+50-89-12).

The sum of changes at hour 2 is 29 (-15+28+71-55).

The sum of changes at hour 3 is -8 (-15+29+54-76).

The sum of changes at hour 4 is 91 (52-60+41+58).


However, a runtime error was found, and the program must be modified.

Correct the following program error.


 [Input Format]

The first line is N that specifies the measured days and hours (measured days = measured hours = N). 

Beginning with the second line, N data values are input in each of N rows.


The valid range of N is 1<N<25.

The change in fine dust is positive if the measured value exceeds the reference value and negative if the measured value is below the reference value. It is an integer in the range of -100,000,000–100,000,000.


The value in a row means the measured value at hour N on the day.


[Output Format]

Output the day that shows the largest sum of hourly fine dust changes and the hour that shows smallest sum of hourly changes over N measured days with a space between the values as the separator. If there are largest values or smallest values that are the same, output the earlier date or time.
