MC Company's G6 Development Team is developing a new calculator program for smartphones.

Senior Research Engineer M.C. Kim who is responsible for the code is on a business trip, and you are told to read the current code and complete the calculator program development.



[Requirement]


The calculator program uses the infix notation as shown below for input of equation. The program supports only the addition and multiplication operations, and the multiplications has priority for calculation when the equation has both addition and multiplication.




Infix notation uses operators placed between operands like “1 + 2”. Most of binary operations use the infix notation.



 [Input Format]


First line: Integer N (2<=N<=50000) which represents the number of terms of the input equation

Second line: Equation in the form of A1 OP1 A2 OP2 … AN-1 OPN-1 AN

The integer Ai (1<=Ai<=999) and the operator OPi (+ or *) are separated by a space.



[Output Format]


Output the remainder of the result of equation divided by 10,000.
